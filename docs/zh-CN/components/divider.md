## 基本用法

```schema: scope="body"
{
    "type": "divider"
}
```

## 不同样式

```schema: scope="body"
[
    {
        "type": "divider"
    },
    {
        "type": "divider",
        "lineStyle": "solid"
    }
]
```

## 属性表

| 属性名    | 类型     | 默认值     | 说明                                |
| --------- | -------- | ---------- | ----------------------------------- |
| type      | `string` |            | `"dialog"` 指定为 Dialog 渲染器     |
| className | `string` |            | 外层 Dom 的类名                     |
| lineStyle | `string` | `"dashed"` | 分割线的样式，支持`dashed`和`solid` |
