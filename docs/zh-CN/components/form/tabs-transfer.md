## 属性表

更多配置请参考[穿梭器（Transfer）](./transfer)。

## 事件表

| 事件名称   | 事件参数                              | 说明   |
| ---------- | ------------------------------------- | ------ |
| change     | `value: string` 时间值                | 值变化 |
| tab-change | `key: number` 当前选择的 tab 的 index | 值变化 |

## 动作表

| 动作名称  | 动作配置 | 说明   |
| --------- | -------- | ------ |
| clear     | -        | 清空   |
| reset     | -        | 值重置 |
| selectAll | -        | 全选   |
