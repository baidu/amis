## 属性表

当做选择器表单项使用时，除了支持 [普通表单项属性表](./formitem#%E5%B1%9E%E6%80%A7%E8%A1%A8) 中的配置以外，还支持下面一些配置

| 属性名   | 类型      | 默认值  | 说明               |
| -------- | --------- | ------- | ------------------ |
| half     | `boolean` | `false` | 是否使用半星选择   |
| count    | `number`  | `5`     | 共有多少星可供选择 |
| readOnly | `boolean` | `false` | 只读               |
