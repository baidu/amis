## 属性表

## 事件表

| 事件名称 | 事件参数               | 说明                 |
| -------- | ---------------------- | -------------------- |
| change   | `value: string` 时间值 | 值变化               |
| focus    | -                      | 获得焦点(非内嵌模式) |
| blur     | -                      | 失去焦点(非内嵌模式) |

## 动作表

| 动作名称 | 动作配置 | 说明   |
| -------- | -------- | ------ |
| clear    | -        | 清空   |
| reset    | -        | 值重置 |
