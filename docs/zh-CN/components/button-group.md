## 基本用法

```schema: scope="body"
{
  "type": "button-group",
  "buttons": [
    {
      "type": "button",
      "label": "Button",
      "actionType": "dialog",
      "dialog": {
        "confirmMode": false,
        "title": "提示",
        "body": "对，你刚点击了！"
      }
    },

    {
      "type": "button",
      "actionType": "url",
      "url": "https://www.baidu.com",
      "blank": true,
      "label": "百度一下"
    },

    {
      "type": "button",
      "label": "普通按钮"
    }
  ]
}
```

## 垂直模式

配置`"vertical": true`，实现垂直模式

```schema: scope="body"
[
  {
    "type": "button-group",
    "vertical": true,
    "buttons": [
      {
        "type": "button",
        "label": "按钮1"
      },
      {
        "type": "button",
        "label": "按钮2"
      },
      {
        "type": "button",
        "label": "按钮3"
      }
    ]
  }
]
```

## 属性表

| 属性名    | 类型                      | 默认值           | 说明                       |
| --------- | ------------------------- | ---------------- | -------------------------- |
| type      | `string`                  | `"button-group"` | 指定为 button-group 渲染器 |
| className | `string`                  |                  | 外层 Dom 的类名            |
| buttons   | Array<[Action](./action)> |                  | 行为按钮组                 |
| vertical  | `string`                  |                  | 是否使用垂直模式           |
