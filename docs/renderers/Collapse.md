### Collapse

折叠器

-   `type` 请设置成 `collapse`
-   `title` 标题
-   `collapsed` 默认是否要收起。
-   `className` CSS 类名，默认：`bg-white wrapper`。
-   `headingClassName` 标题 CSS 类名，默认：`font-thin b-b b-light text-lg p-b-xs`。
-   `bodyClassName` 内容 CSS 类名。

```schema:height="350" scope="body"
{
    "type": "collapse",
    "title": "标题",
    "body": "内容。。。"
}
```
