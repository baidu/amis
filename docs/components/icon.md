## 基本使用

```schema:height="300"
{
    "type": "page",
    "body": {
        "type": "icon",
        "icon": "cloud"
    }
}
```


| 属性名    | 类型     | 默认值 | 说明                           |
| --------- | -------- | ------ | ------------------------------ |
| type      | `string` | `icon` | 指定组件类型                   |
| className | `string` |        | 外层 CSS 类名                  |
| icon      | `string` |        | icon 名，只支持 fontawesome v4 |
