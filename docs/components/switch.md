## 基本用法

```schema:height="300"
{
    "type": "page",
    "body": {
        "type": "switch",
        "value": true
    }
}
```


| 属性名    | 类型     | 默认值 | 说明                            |
| --------- | -------- | ------ | ------------------------------- |
| type      | `string` |        | `"switch"` 指定为 Dialog 渲染器 |
| className | `string` |        | 外层 Dom 的类名                 |
| trueValue | any      |        | 真值，当值为该值时，开关开启    |
| option    | `string` |        | 右侧选项文本                    |
