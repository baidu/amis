## 属性表

当做选择器表单项使用时，除了支持 [普通表单项属性表](./formitem#%E5%B1%9E%E6%80%A7%E8%A1%A8) 中的配置以外，还支持下面一些配置

| 属性名    | 类型                            | 默认值 | 说明                 |
| --------- | ------------------------------- | ------ | -------------------- |
| min       | [模板](../../concepts/template) |        | 最小值               |
| max       | [模板](../../concepts/template) |        | 最大值               |
| step      | `number`                        |        | 步长                 |
| precision | `number`                        |        | 精度，即小数点后几位 |
