# 说明

## 原理

docx 渲染器，原理是将 docx 里的 xml 格式转成 html

相对于 Canvas 渲染，这个实现方案比较简单，最终页面也可以很方便复制，但无法保证和原始 docx 文件展现一致，因为有部分功能难以在 HTML 中实现，比如图文环绕效果。

## 已知不支持的功能

- 艺术字
- 域
- 对象
- wmf，需要使用 https://github.com/SheetJS/js-wmf
